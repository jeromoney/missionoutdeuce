# MissionOut Responder

This folder now contains the starter Flutter responder app with web support.

Current responsibilities:

- Receive mission alerts on iOS and Android
- Provide a browser-based responder view through Flutter web
- Bridge Flutter UI to native alert handling
- Own the phone alarm, wake, and full-screen alert path
- Sync responder actions back to the backend

Planned stack:

- Flutter for shared mobile UI
- Flutter web for responder browser access
- Kotlin for Android native alert handling
- Swift for iOS native alert handling
