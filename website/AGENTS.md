# Website Scope

This directory is owned by the Marketing / Website thread.

## Purpose

Own the public website, brochure pages, and non-product web content.

## Responsibilities

- landing pages
- brochure content
- public branding pages
- static website structure

## Constraints

- Do not modify backend or product application logic unless explicitly instructed.
- Do not turn the website into the dispatcher, responder, or Team Management app.
- Keep public-site concerns separate from operational product workflows.

## Adjacent Ownership

- Product Flutter clients belong in `../UserInterface/`
- Backend API belongs in `../backend/`
- DNS and hosting decisions belong to the Infrastructure / Deploy thread
